# 👨‍💻 Landing Portfolio App

Welcome to the **Landing Portfolio App**! This project is a modern, responsive personal portfolio built with **Angular 17**, designed to showcase projects, skills, and professional experience in a sleek and elegant way.

---

## ✨ Features

- 🚀 **Modern Tech Stack**: Built with Angular 17 and Server-Side Rendering (SSR).
- 🎨 **Responsive Design**: Mobile-first approach using Bootstrap and custom styling.
- 🧩 **UI Components**: Enhanced with Angular Material for a premium look and feel.
- 🔠 **Iconography**: Integrated with FontAwesome for beautiful, scalable icons.
- ⚡ **Performance**: Optimized build with fast load times and smooth animations.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have Node.js and Angular CLI installed.
- npm
  ```sh
  npm install npm@latest -g
  ```
- Angular CLI
  ```sh
  npm install -g @angular/cli
  ```

### Installation

1. Clone the repository
   ```sh
   git clone https://github.com/Zifrikc/AVPortafolio.git
   ```
2. Navigate into the project directory
   ```sh
   cd AVPortafolio
   ```
3. Install NPM packages
   ```sh
   npm install
   ```

---

## 💻 Development Server

Run `npm run start` or `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The application will automatically reload if you change any of the source files.

### 🏗️ Build

Run `npm run build` or `ng build` to build the project. The build artifacts will be stored in the `dist/` directory.

### 🧪 Testing

- Run `npm run test` or `ng test` to execute the unit tests via [Karma](https://karma-runner.github.io).

---

## 🌐 Server-Side Rendering (SSR)

This project includes SSR capabilities out-of-the-box for better SEO and initial load performance.
To test the SSR build locally:
```sh
npm run serve:ssr:landingPortafolioApp
```
